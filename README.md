# MYR Transportes SpA - Sitio Web

Sitio web corporativo para MYR Transportes SpA, empresa de transporte de carga terrestre con cobertura nacional en Chile.

## Descripción

Landing page y sistema de cotización para servicios de transporte de carga de Arica a Punta Arenas.

## Stack Tecnológico

- **Framework:** Astro
- **Estilos:** Tailwind CSS
- **Iconos:** Material Symbols
- **Fuentes:** Plus Jakarta Sans, Manrope

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Estructura

```
src/
├── components/     # Componentes reutilizables
├── layouts/        # Layout principal
├── pages/          # Páginas del sitio
│   ├── index.astro
│   ├── cotizar.astro
│   └── servicios/
└── styles/         # Estilos globales
public/
└── img/            # Imágenes del sitio
```
